  • Crossstitch
  • Ovotovata
  • Lohkous
  • Chained Block チェンブロ
  • Canal View
  • Choco Banana チョコバナナ
  • Building Walk ビルウォーク
  • Voxas
  • One Room One Door ワンルームワンドア
  • Tonttiraja
  • Rassi Silai
  • Parquet
  • La Paz
  • Tren パーキング
  • Pentominous
  • Hinge ちょうつがい
For other puzzle types and a detailed manual, please visit puzz.link.

Terms, Privacy

Puzzle content

For all puzzles shared via this site, the copyright lies with their authors. We merely serve an interface for solving those puzzles; any issues should be taken up with the site hosting the link.

Privacy

The solving applet stores user preferences in the browser local storage. These preferences are not shared with any servers.

Contact

This fork is maintained by X_Sheep. You can report issues specific to this fork on the issue tracker.
